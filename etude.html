C'est quoi HTML en informatique ?

  • L'HyperText Markup Language, HTML,dans sa dernière version, HTML5, désigne un type de langage informatique descriptif. Il s'agit plus précisément d'un format de données utilisé dans l'univers d'Internet pour la mise en forme des pages Web. Il permet, entre autres, d'écrire de l'hypertexte, mais aussi d'introduire des ressources multimédias dans un contenu.
  • Un document HTML est un fichier qui contient du texte et enregistré avec l'extension ㅤ.htmlㅤ
  • Chaque code html doit avoir une structure spéciale et chaque balise doit être compris entre < > comme cet exemple:
  • Il commence par le balise < !DOCTYPE html >
  • La page se trouve entre la balise <html> ....... < /html>
  • Un document html est composé de:
    • <head> ....... < /head>
    • <body> ....... < /body>

ㅤㅤ< /body>
< /html>
