HTML

C'est quoi HTML en informatique ?

  • L'HyperText Markup Language, HTML,dans sa dernière version, HTML5, désigne un type de langage informatique descriptif. Il s'agit plus précisément d'un format de données utilisé dans l'univers d'Internet pour la mise en forme des pages Web. Il permet, entre autres, d'écrire de l'hypertexte, mais aussi d'introduire des ressources multimédias dans un contenu.
  • Un document HTML est un fichier qui contient du texte et enregistré avec l'extension ㅤ.htmlㅤ
  • Il commence par le balise < !DOCTYPE html >
  • La page se trouve entre la balise <html> ....... < /html>
  • Un document html est composé de:
    • <head> ....... < /head>
    • <body> ....... < /body>

Balises de premier niveau:

Les balises de premier niveau sont les principales balises qui structurent une page HTML. Elles sont indispensables pour réaliser le « code minimal » d'une page web.

Balise Description
html Balise principale
head En-tête de la page
body Corps de la page

Code minimal d'une page HTML :

< !DOCTYPE html >
<html>
ㅤㅤ<head>
ㅤㅤㅤㅤ<meta charset="UTF-8">
ㅤㅤㅤㅤ<title> test < /title>
ㅤㅤ< /head>
ㅤㅤ<body>
ㅤㅤㅤㅤ<p> Welcome to my world < /p>
ㅤㅤ< /body>
< /html>

Balises d'en-tête:

Ces balises sont toutes situées dans l'en-tête de la page web, c'est-à-dire entre <head> et < /head> :

Balise Description
meta Métadonnées de la page web (charset, mots-clés, etc.)
title Titre de la page
